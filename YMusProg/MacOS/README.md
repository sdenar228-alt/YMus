# YMus Desktop для macOS

Разработка версии для macOS начинается после стабилизации Windows MVP.

Планируемый общий стек: Tauri 2, Rust, React и TypeScript. Интерфейс и серверный
протокол релизов должны быть общими с Windows-версией, а системные адаптеры
обнаружения браузеров, применения политик и установки пакетов будут отдельными.

Для публичного распространения потребуется Apple Developer ID, подпись
приложения и notarization. Тихая установка расширений вне магазинов ограничена
политиками браузеров и, как правило, требует MDM или управляемого устройства.
